About the Web

The web is a vast network of interconnected documents and resources, accessible via the Internet. It was invented by Tim Berners-Lee in 1989 and has since become an essential part of modern life. The web is built on top of the Internet, which is a global network of interconnected computers and servers.

The web is made up of billions of web pages, each with its own unique address, known as a URL. These web pages are linked together using hyperlinks, which allow users to navigate from one page to another. The web is also home to a vast array of online services, including social media, online banking, and e-commerce sites.
